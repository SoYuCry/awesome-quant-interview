# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This is an **awesome-list** style repository for quantitative finance interview preparation, written in Chinese. It consists of a single `README.md` file (~1400 lines) containing curated resources: open-source tools, strategy papers, book recommendations, and interview Q&A ("八股文").

There is no build system, no tests, and no application code.

## Content Structure

The README is organized into four major sections:

1. **开源工具** - Curated links to quant tools (backtesting, platforms, AI+finance, libraries)
2. **策略方向与论文** - Academic papers organized by strategy type (CTA, HFT, market-making, stat-arb, options/vol)
3. **书单与资源** - Books, data sources, brokerages, communities, competitions
4. **面试八股文** - Interview Q&A covering math/stats, programming (Python/C++), factor/alpha strategies, and ML/DL

## Conventions

- Uses star emojis (🌟/🌟🌟) to indicate importance/recommendation level
- Book references use 📖 emoji
- Each strategy section has a blockquote intro explaining the category
- Interview Q&A follows a consistent format: question heading, "标准答案" (standard answer), then "面试加分点" (bonus points)
- LaTeX math notation is used throughout the interview section
- All content is in Chinese; tool/paper names stay in English
- Links follow the awesome-list format: `[Name](URL) - Description`
- Tables are used for books, data sources, and broker research teams
